# flutter_template

A new Flutter project.

## Getting Started

1. Clone this repo

2. create a project

    ```dash
        flutter create --project-name your_app_name .
    ```

3. generate localizations files

    ```dash
        flutter gen-l10n
    ```

4. replace each "package:flutter_feature_first_template/..." with your project name

5. check the TODOs
